### :wrench: Como Utilizar

1. Fork ou clone esse repositório:
```sh 
git@github.com:Lorenalgm/nlw.git
```
2. Abra o projeto no seu editor de texto
3. Entre na pasta frontend e inicie o server!
```sh 
    cd frontend
    yarn start
```
